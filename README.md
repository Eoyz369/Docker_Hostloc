# Docker_Hostloc


### 使用方法
#### 一、新建`.env`文件，编辑以下内容并保存
```bash
# Telegram Bot 的 API Token
TOKEN = config["BOT_TOKEN"]
# Telegram Channel 的 ID
# CHANNEL_ID为简洁模式，CHANNEL_ID_2为完整推送
CHANNEL_ID = config["CHANNEL_ID"]
CHANNEL_ID_2 = config["CHANNEL_ID"]
```
